## Svelte 3.22

### Svelte component

```.js
<script>
  let items = [];
  function addItem() {
    const newItem = Math.round(Math.random() * 100);
    items = [...items, newItem];
  }
</script>

<style>
  h2 {
    color: darkblue;
  }
</style>

<div>
	<h2>Array Demo</h2>
	<button on:click={addItem}>Add Item</button>

  {#if items.length > 0}
    <h3>List items:</h3>
    <ul>
      {#each items as val, idx}
        <li>{val}</li>
      {/each}
    </ul>
  {/if}
</div>
```
